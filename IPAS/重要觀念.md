#
```
機密性(Confidentiality)
資訊不得被未經授權之個人、實體或程序所取得或揭露的性質
```
```
完整性(Integrity)
對資訊之精確與完整安全保證的性質
```
```
可用性(Availability )
已授權實體在需要時可存取與使用資訊之性質
```
```
鑑別度(Authenticity)
可證明一主體或資源之識別就是其所聲明者的特性。鑑別性適用於如使用者、程序、系統與資訊等實體
```
```
不可否認性(Non-repudiation)
對一已發生之行動或事件的證明，使該行動或事件往後不能被否認的能力
```
```
可靠度(Reliability)
始終如一預期之行為與結果的性質
```
```
可歸責性(Accountability)
確保實體之行為可唯一追溯到該實體的性質
```
